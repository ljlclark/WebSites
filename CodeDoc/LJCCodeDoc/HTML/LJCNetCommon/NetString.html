public class NetString

Contains common string related static functions.

Remarks

The NetCommon class contains string related static helper functions for all .NET program types.

Checking String Values
These include: HasValue(), IsDigits() and IsEqual().

Formatting a String
These include: GetPropertyName(), ExceptionString() and InitString().

Parsing Delimited String Values
These include: GetDelimitedAndIndexes(), GetDelimitedString() and GetStringWithDelimiters().

Soundex Functions
These are helper functions for creating Soundex values.
These include: CreateLSoundex(), CreatePSoundex(), IsSoundexLetter() and Phonetic().
