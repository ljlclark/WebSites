Class ControlColumn

Syntax

C#
public class ControlColumn

Represents a column of Controls consisting of ControlRows.

Remarks

A ControlColumn is a definition of the logically contained Labels column and Controls columns. These are defined by the LabelsWidth and ControlsWidth properties.

Index - A unique value which defines the relative position of the ControlColumn in the Dialog.

LabelsWidth - The width of the widest Label in the Labels column. The Label width is calculated from the Label Text value. The Label controls are left aligned to the Dialog BorderHorizontal value.

ControlsWidth - The width of the widest data Control in the Controls column. This is usually a TextBox control. The Control width is calculated from the Control MaxLength property. The controls are left aligned to the associated Labels column.

RowCount - Contains the count of the ControlRows in that ControlColumn.

Properties
ControlsWidth Gets or sets the Controls column width.
Index Gets or sets the unique Index value.
LabelsWidth Gets or sets the Labels column width.
RowCount Gets or sets the number of ControlRows.
Width Gets the ControlColumn width.
