Property FirstRowIndex

public Int32 FirstRowIndex

Gets or sets the grid FirstRowIndex value.

Remarks

The FirstRowIndex value indicates the row that should be positioned at the top of the list. It works with the CurrentRowIndex property to maintain the current position when switching between pages.
