Assembly LJCDBViewDAL

Provides Data Access Layer (DAL) classes for the LJCDBViewDAL library.

The LJCDBViewDAL library contains Data and Manager classes for presenting an object oriented view of the Data Views database data.

The Manager classes use LJC.DataAccess to communicate with the database.
