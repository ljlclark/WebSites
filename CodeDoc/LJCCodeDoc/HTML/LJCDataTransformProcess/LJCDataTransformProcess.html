Assembly LJCDataTransformProcess

The Transform Process library provides classes to execute Data Transform Processes.

A Transform DataProcess contains Steps which are defined in the Transform Process Data. Each Step in the DataProcess is executed in Sequence order.

A Transform Process Step contains StepTasks which are defined in the Transform Process Data. Each StepTask in the Step is executed in Sequence order.

The LJC.DataTransformProcess program is part of the LJC.DataTransform solution.
