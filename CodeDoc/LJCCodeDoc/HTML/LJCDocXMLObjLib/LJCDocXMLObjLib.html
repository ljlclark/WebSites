Assembly LJCDocXMLObjLib

The Documentation "Doc" classes.
These classes contain the deserialized XML documentation data. This data is used in the "Data" classes.

The "Doc" member classes include: Doc, DocAssembly, DocMembers, DocMember, Params(DocParams), DocParam, Paras(DocParas), DocPara, Remarks(DocRemarks) and Example(DocExample).

The LJCDocXMLObjLib project is part of the LJC.DocLib solution.
