Assembly LJCGenTextEdit

The GenText Edit Utility is used to test GenText Templates and GenText XML Data files.

After the user selects a template, data file and output file, they can generate the output file code. The templates, data files and output files can be created, modified and saved within the editor.

The template and output editing is simple and is intended for minor changes. An editor or IDE for the source file type is recommended for creating a new template or major changes.

The LJCGenTextEdit project is part of the LJCGenText solution.
