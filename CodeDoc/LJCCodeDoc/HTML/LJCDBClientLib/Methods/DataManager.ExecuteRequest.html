public DbResult ExecuteRequest(DbRequest dbRequest)

Executes the supplied request.

Remarks

The DataManager can communicate with one of three different Data Access objects.

It first checks the DbServiceRef.DbDataAccess property for an instance of the DbDataAccess object. If the DbDataAccess property is set, it uses this reference for data access.

If the DbDataAccess property is null, then it checks the DbServiceRef.DbService property for an instance of the DbService object. If the DbService property is set, it uses this reference for data access.

If the DbDataAccess and DbService properties are null, then it checks the DbServiceRef.DbServiceClient property for an instance of the DbServiceClient object. If the DbServiceClient property is set, it uses this reference for data access.
