public class TextPagination

Remarks

The TextPagination Initial Load Process

The Form that includes the LJCDataGrid control, contains a private IDataSource or DataManager object. This DataManager object is initialized and the PageSize set before the LJCDataGrid columns are configured.

The LJCDataGrid columns must be configured before attempting to display any data in the LJCDataGrid control.

The Form also includes a private TextPagination object. This objects is initialized before attempting to display any data in the LJCDataGrid control.

After an instance of the TextPagination object is created, an Event Handler is created for the TextPagination.DataRetrieve event and the SelectCurrentRow event. Then the TextPagination.PageSize is set and the TextPagination.Initialize() method is called.

The TextPagination.Initialize() method fires the DataRetrieve event. The Form.DataRetrieve event handler saves the TextPagination.PageStartIndex and PageSize to the IDataSource or DataManager properties of the same name. It then calls the Form private DataRetrieve() method.

The List Form DataRetrieve() method calls the DataSource or DataManager Load() method. This method uses the DataManager PageStartIndex and PageSize values to load the next or previous pages of data. The DataRetrieve() method shows the first page of data for the initial load.

The Paging Process

The PageGridProcess() method is called when the user changes the selected item in the list. It calls the TextPagination.PageUp() method if the selected record is in the top 10 records of the list and pagination is not at the beginning of the data. The TextPagination FirstRowIndex and CurrentRowIndex are saved from the current position of the grid.

The PageGridProcess() calls the TextPagination.PageDown() method if the selected record is in the last 10 of the list and not at the end of the data. The TextPagination FirstRowIndex, CurrentRowIndex and RowCount are set from the current position and content of the grid.

As the user pages through the data, the TextPagination PageUp() and PageDown() methods attempt to include the old previous or next page of data so that two pages of data are displayed. This provides the user with better data context as they can always see the rows of data on both sides of a page break.

The TextPagination PageUp() and PageDown() methods also calculate the new FirstRowIndex and CurrentRowIndex values. Then they call the form SelectCurrentRow event which sets those values. This prevents the list from appearing to move which preserves the current item context and provides a smoother visual experience for the user.
