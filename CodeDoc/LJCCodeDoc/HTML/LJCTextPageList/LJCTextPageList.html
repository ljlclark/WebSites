Assembly LJCTextPageList

An End-to-End Pagination Process is by necessity fairly complicated. This is because there are several objects involved. These are the Datasource Manager, Pagination Object and List Control.

A Datasource Manager is responsible for interacting with the Datastore. This may be a DBMS Table, a Disk File, a Data Service or some other source of data.

The PageStartIndex and PageSize values are added to the Datasource Manager. These values are used for retrieving the next or previous pages of data.

The Pagination Object performs the task of determining the records that appear in the page by setting the PageStartIndex and PageSize values. It also determines if the list should be cleared for the next or previous pair of pages.
