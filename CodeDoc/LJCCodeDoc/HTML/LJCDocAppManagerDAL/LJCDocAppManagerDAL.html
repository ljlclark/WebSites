Assembly LJCDocAppManagerDAL

Provides Data Access Layer (DAL) classes for the LJCDocAppManager program.

The LJCDocAppManagerDAL library contains Data and Manager classes for presenting an object oriented view of the DocApp Manager database data.

The LJCDocAppManagerDAL project is part of the LJCDocAppManager solution.
