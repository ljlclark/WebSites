public class DataDetailDialog : Form

The DataDetail Dynamic Detail dialog.

Remarks

The DataDetailDialog form is designed to dynamically create a Detail Dialog to allow display and edit of Data Record values which are contained in the LJCRecord (DbColumns) collection.

The Dialog contains one or more TabControl pages. Each TabPage can consist of multiple ControlColumn objects. Each ControlColumn object can consist of multiple ControlRow objects.
