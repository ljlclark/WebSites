public static States LJCDeserialize(out String& errorText, String fileSpec = null)

Deserializes from the specified XML file.

Remarks

If the requested XML file does not exist, a default file will be created with the fifty U.S. states, plus American Samoa(AS), District of Columbia(DC), Federated States of Micronesia(FM), Guam(GU), Marshall Islands(MH), Northern Mariana Islands(MP), Palau(PW), Puerto Rico(PR) and Virgin Islands(VI).

A default StateLookup XML file will also be created if it does not exist.
