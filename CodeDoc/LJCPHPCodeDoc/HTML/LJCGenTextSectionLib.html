LJCGenTextSectionLib.php

The GenText Section Library

Remarks

Contains the GenData classes used in the text generation process. These classes are also used to Serialize and Deserialize the GenData XML that defines the text Sections, Items and Replacement values.

GenData is the XML data format for the data that is merged with a text template file to generate a customized output file.

See: LJCSections Deserialize() method for the layout of the GenData XML.

Classes
LJCDirective Represents a template Directive.
LJCSection Represents a template Section.
LJCSections Represents a collection of Section objects.
LJCItem Represents a Section Item.
LJCReplacement Represents Item Replacements.
