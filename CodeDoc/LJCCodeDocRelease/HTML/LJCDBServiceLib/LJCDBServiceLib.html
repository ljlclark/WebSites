Assembly LJCDBServiceLib

Data Service Library

The Data Service library contains classes, which receive Request XML messages. These messages are deserialized into Message Objects, which are passed to the LJCDBDataAccessLib methods to perform the requested data operations.

The data is returned to the calling program as a Result XML message.
