Class ControlRow

public class ControlRow : IComparable<ControlRow>

Represents the controls for a DataValue.

Remarks

A ControlRow is a definition of the contained RowLabel and RowControl controls. A ControlRow is created for each Data Record value. The controls in a ControlRow are top aligned based on the Dialog BorderVertical, ControlRowSpacing and ControlRowHeight values.

DataValueName - The name of the Data Record value in the DataValues collection. The DataValues property is a DbColumns collection object.

RowLabel - A reference to the row Label control.

RowControl - A reference to the row data control.

TabPageIndex - The TabPage where the ControlRow is displayed.

ColumnIndex - The ControlColumn where the ControlRow is displayed.

RowIndex - The row index where the Label and data Controls are displayed.

TabIndex - The tabbing index for the Label control in the ControlRow.
