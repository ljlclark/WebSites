Assembly LJCDataTransformDAL

Provides Data Access Layer (DAL) classes for the LJC.DataTransform programs.

The LJCDataTransformDAL library contains Data and Manager classes for presenting an object oriented view of the Data Transform database data.

The Manager classes use LJC.DataAccess to communicate with the database.

The DataTransformDAL library is part of the LJC.DataTransform solution.
