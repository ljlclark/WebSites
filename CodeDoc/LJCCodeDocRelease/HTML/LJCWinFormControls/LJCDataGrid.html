Class LJCDataGrid

Syntax

C#
public class LJCDataGrid : DataGridView

Provides custom functionality for a DataGridView control. (D)

Remarks

The Row Data Methods
The grid has extended methods for managing rows and row data.
These methods are LJCRowsClear(), LJCAddRow(), LJCInsertRow() and LJCRowSetValues().

The Row Selection Methods
The grid has extended methods for managing the current row.
These methods are LJCGetMouseRow(), LJCSetMouseCurrentRow() and LJCSetCurrentRow().

The Row Selection Changed Methods
There are extended methods for checking if the selected row has changed.
These methods are LJCIsDifferentRow() and LJCSetLastRow().

The Grid Configuration Methods
The grid has extended methods for grid configuration.
These methods are LJCSetPlain(), LJCAddColumn(), LJCAddDisplayColumns() and LJCRestoreColumnValues().
