Assembly LJCDocLibDAL

Provides Data Access Layer (DAL) classes for the LJCDocGenLib library.

The LJCDocLibDAL library contains Data and Manager classes for presenting an object oriented view of the DocGroup XML data.

The LJCDocLibDAL project is part of the LJC.DocLib solution.
